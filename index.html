SASS drop shadows without images

A bunch of CSS classes and SASS mixins for creating resolution-independent crisp and visually appealing shadows without using any images

It's really simple to use these. Just add the CSS class or apply the mixin. For documentation head over to the Github page …

Default

CSS Class: .shadow

SASS mixin: @include shadow();

Perspective

CSS Class: .shadow.perspective

SASS mixin: @include shadow(perspective);

Raised

CSS Class: .shadow.raised

SASS mixin: @include shadow(raised);

curved

CSS Class: .shadow.curved

SASS mixin: @include shadow(curved);

curved top

CSS Class: .shadow.curved-top

SASS mixin: @include shadow(curved-top);

curved bottom

CSS Class: .shadow.curved-bottom

SASS mixin: @include shadow(curved-bottom);

Vertical curved

CSS Class: .shadow

SASS mixin: @include shadow();

Vertical curved top

CSS Class: .shadow

SASS mixin: @include shadow();

Vertical curved bottom

CSS Class: .shadow

SASS mixin: @include shadow();

Rotated box

CSS Class: .shadow

SASS mixin: @include shadow();
